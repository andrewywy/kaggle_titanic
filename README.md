# kaggle_titanic
Predict whether people die or survive, model evaluation, feature importance finding
Involve:
1. EDA, e.g. check null, mean, median,
   relationship between predict value and features -> indentify strong predictors before training
3. Data Handling, e.g. fill null, drop columns
4. Engineering, e.g. add columns as features
5. Encoding, scaling
6. Split train/test group
7. train logistic regression model, random forest -> choose random forest
8. Model evaluation(Cross validation, confusion matrix, classification report)
9. model interpretation
